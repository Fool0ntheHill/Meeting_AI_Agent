## 1. 用户认证模块

### 1.1 登录功能
- ⏳ 登录页面 UI
- ⏳ 用户名输入表单
- ⏳ 调用登录 API (`POST /api/v1/auth/dev/login`)
- ⏳ Token 存储到 localStorage
- ⏳ 登录成功跳转到任务列表
- ⏳ 登录失败错误提示

### 1.2 Token 管理
- ⏳ Token 自动过期检测
- ⏳ Token 过期自动跳转登录
- ⏳ 请求拦截器自动添加 Authorization 头
- ⏳ 401 错误自动清除 Token

### 1.3 退出登录
- ⏳ 退出按钮
- ⏳ 清除 Token
- ⏳ 跳转到登录页

- **Quill**
- **TinyMCE**
